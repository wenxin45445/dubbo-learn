## 说明

### 运行项目

- 直接用游览器打开*public*文件夹下的*index.html*即可
- 或者也可以在控制台输入`node index.js`启动一个静态文件服务器，然后在游览器地址栏输入`http://127.0.0.1:8080/`打开网页
- *src*目录下是源码文件，无法再游览器上运行，*public*目录下的是通过**webpack**打包之后生成的静态网页文件，能够在游览器上运行
